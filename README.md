🌙 Daytick – Habit Tracker

Daytick is a clean and modern habit tracking web application designed to help users build consistency and stay productive.
It provides a simple dashboard to track daily habits, monitor progress, and maintain motivation.

✨ Features
📊 Daily Progress Tracking – Shows percentage of habits completed.
✅ Habit Checklist – Mark habits as completed for the day.
📅 Weekly Consistency View – Helps users maintain streaks.
🗂 Habit Categories – Organize habits like Health, Study, and Fitness.
🎨 Modern Dark Dashboard UI
➕ Add New Habits Easily
🖥 Preview

Daytick dashboard includes:

Greeting header with date
Habit progress cards
Habit list with categories
Sidebar navigation (Dashboard, History, Profile, Feedback)

🛠 Tech Stack
HTML5
CSS3
JavaScript
